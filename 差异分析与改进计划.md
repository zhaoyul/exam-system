# 中广核职业技能等级认定系统 - 差异分析与改进计划

## 一、实际系统功能架构

### 从 yqdjrd.zyjndj.com 登录页轮播图提取的八大中心：

| 中心 | 功能描述 |
|------|---------|
| **备案中心** | 负责集团公司职业技能等级评价机构在人力资源社会保障部备案 |
| **监管中心** | 对集团公司下属分支机构的职业技能等级评价活动进行监督和管理，规范指导分支机构开展评价活动 |
| **题库中心** | 集团公司组织命题专家按国家职业技能标准结合集团公司生产实际开发统一标准的评价题库，为分支机构提供评价试卷 |
| **证书中心** | 集团公司对职业技能等级评价证书实行统一管理统一发放，做到全集团公司评价证书联网通用 |
| **溯源中心** | 对职业技能等级评价活动从评价计划、考生报名、考场安排、组织实施、成绩管理到证书颁发，进行全过程溯源调查 |
| **标准化中心** | 国家职业技能标准的制修订管理 |
| **评价中心** | 技能等级评价活动管理 |
| **考试中心** | 考试组织实施管理 |

### 从 jineng.cnnc.com.cn（中核集团类似系统）提取的功能：

**集团监管端：**
- 集体报名、考场控制、省级备案、等级认定、认定申报

**个人中心（考生端）：**
- 个人网报、打印准考证、智能化考试、成绩查询、证书查询

## 四、改进计划

### Phase 1: 核心功能补充（最高优先级）

#### 4.1 新增溯源中心模块
实现从评价计划到证书颁发的全链条追溯：
- **溯源查询** - 输入身份证号/证书编号查询完整认定链路
- **链路可视化** - 以时间线/流程图形式展示每个环节状态
- **异常标记** - 自动标记链路中的异常环节（成绩缺失、证书未发等）
- **报表导出** - 导出完整溯源报告

#### 4.2 新增考务监控看板
- **实时考试监控** - 展示当前进行中的考试状态
- **考生到场率** - 实时统计各考场考生到场情况
- **异常情况预警** - 缺考、违规等异常实时告警
- **数据刷新机制** - 定时刷新考试数据

#### 4.3 新增站内消息/通知中心
- **消息列表** - 收发系统消息
- **通知设置** - 配置消息推送规则
- **消息标记** - 已读/未读状态管理

### Phase 2: 考生端（个人中心）开发

#### 4.4 新增个人中心模块
- **个人网报** - 考生自主填写报名信息
- **准考证打印** - 生成并打印准考证
- **在线考试** - 理论考试在线答题界面
- **成绩查询** - 个人成绩查询
- **证书查询** - 个人证书查询和验证

### Phase 3: 界面优化

#### 4.5 登录界面改造
- 使用核电站实景大图作为背景
- 添加系统模块轮播展示
- 支持两种登录方式切换
- 添加系统备案标识

#### 4.6 模块命名优化
- 侧边栏菜单按"中心"概念重组
- 统一模块命名风格

## 五、下一步建议

考虑到实际系统的规模和复杂度，建议按以下顺序实施：

1. **溯源中心** - 这是管理端最核心的缺失功能，直接体现系统管理能力
2. **考务监控看板** - 增强考试过程的可视化管理
3. **站内消息中心** - 提升系统协作能力
4. **登录界面改造** - 提升系统专业形象
5. **个人中心** - 完整的考生端功能（可作为独立项目开发）
